AI is envolving like nuclear reaction, but industries does not want nuclear bomb, but nuclear plant: a way to turn immense energy into something safe, controllable, reliable and useful in real operation.   

AI 像是核裂变，但是工业界需要的不是原子弹，而是核电站，让核能安全、可控、可靠的在实现运营中有价值的实现。

<2026-06-23>   
AI coding: Stop using AI coding to develop code with old software structure, i.e. old flow, UI/UX, etc.....Instead, develop "AI native" software that have LLM as the core engine, with supportive AI-human interfaces/support, i.e. memory, harness, display/notices, etc.  

AI时代的编程：不是用AI在原来的软件架构下编写代码，复现旧有的工作流，人机界面，操作流程......而是开发AI Native的新软件架构，把AI大模型作为核心功能引擎，辅助以"人-AI"的交互支持(记忆系统，Harness，展示，通知等)

<2026-06-24>  
AI Entrepreneurship is more like manufacturing project than like Internet or Mobile project. Because of the huge upfront investment, the AI LLM giants have financial pressure from day one, so their free subscription/service have been very limited from day one. Token is not free from day one, which forces AI Application Entrepreneurs to think about the product or service cost from day one, just like a manufacturing company. The AI cost increases linearly with delivery, not going marginal like Internet/Mobile projects. AI project need cost optimization from day one.  

因为LLM的巨额前期投入，大模型公司从一开始有财务压力，所以免费服务从一开始就非常有限，Token从一开始就收费，而且收费很高，迫使应用创业公司要像制造业一样思考，因为服务成本会随着客户数量线性提高，没有互联网创业的边际递减效应。技术方案天然需要把优化Token成本作为基础。
